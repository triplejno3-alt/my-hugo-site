从零开始：macOS 快捷指令自动化入门指南

发布于: 2026-04-23

引言

曾几何时，“自动化”这个词听起来像是程序员的专属。但随着 macOS 引入“快捷指令”（Shortcuts），每一个普通用户都拥有了将复杂任务简化的能力。如果你发现自己每天都在重复相同的操作——比如重命名图片、切换工作环境、提取文本——那么这篇文章就是为你准备的。

核心理念：什么是“自动化思维”？

在开始动手之前，我们需要建立一个简单的逻辑模型：输入 -> 动作 -> 输出。

  • 输入：你丢给电脑的东西（如一张图片、一段文字或一个时间点）。
  • 动作：电脑对输入做的处理（如修改尺寸、翻译、发送邮件）。
  • 输出：你想要的结果。

快捷指令的核心，就是通过可视化的模块，把这些“动作”像搭积木一样串联起来。

实战演示：三个改变效率的小例子

1. 一键处理配图：博主必备

处理博客配图时，我们通常需要：压缩、重命名、转为 WebP 格式。

  • 快捷指令逻辑：获取所选文件 -> 调整图像大小 (2000px) -> 设置名称 (日期-项目) -> 转换图像格式 -> 保存。
  • 优势：原本 2 分钟的操作，现在只需右键一按。

2. “专注模式”自动切换

当你点击“开启工作”指令时：

  • 动作：开启“专注模式” -> 打开常用的办公应用 (如 Notion、VS Code) -> 播放指定的音乐列表 -> 隐藏其他无关窗口。
  • 优势：通过仪式感的自动化，快速进入心流状态。

3. 快速生成会议日记

  • 动作：获取当前日期 -> 从日历获取最近的会议标题 -> 创建一个新的 Markdown 笔记 -> 自动填入模板。
  • 优势：不再为整理会议纪要的开头而烦恼。

进阶：如何获取更多高质量指令？

你不需要从头开始发明轮子。

  • 官方中心：快捷指令 App 自带的“中心” (Gallery) 有大量经过苹果官方审核的模版。
  • 社区分享：少数派社区、RoutineHub 等平台有全球开发者分享的复杂工作流。建议先下载现成的指令，通过观察它们的“积木”连接方式来学习逻辑。

总结

自动化不是为了炫技，而是为了把宝贵的时间从琐碎中解放出来，去投入到真正需要人类思考的创造性工作中。现在，打开你的“快捷指令” App，尝试搭建你的第一个积木吧。
